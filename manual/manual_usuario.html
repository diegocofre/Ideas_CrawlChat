Manual de Usuario del Sistema "Crawler GPT"

Descripción del Sistema

El sistema "Crawler GPT" es una aplicación de escritorio diseñada para realizar rastreos web y procesar el contenido utilizando modelos de lenguaje GPT. La aplicación permite a los usuarios iniciar nuevos procesos de rastreo, ver el historial de rastreos anteriores, y gestionar los resultados obtenidos.

Instrucciones de Uso

Menú Principal

  • Nuevo Batch: Inicia un nuevo proceso de rastreo web.
  • Ver Históricos: Muestra el historial de rastreos anteriores.
  • Salir: Cierra la aplicación.

Consola de Batch

  • Consola de Batch: Muestra la salida en tiempo real del proceso de rastreo.
  • Copiar al Portapapeles: Copia el contenido de la consola al portapapeles.
  • Cancelar Batch: Cancela el proceso de rastreo en curso.

Nuevo Batch

  • URL Inicial: Introduce la URL desde donde comenzará el rastreo.
  • Profundidad: Establece la profundidad del rastreo.
  • Sitios: Define el número máximo de sitios a rastrear.
  • Caracteres: Especifica el límite de caracteres a procesar.
  • Guardar e Iniciar: Comienza el proceso de rastreo con los parámetros especificados.

Vista de Batch

  • Detalles del Batch: Muestra los detalles del batch seleccionado, incluyendo la URL inicial, fechas de inicio y fin, y el contenido rastreado.
  • Copiar al Portapapeles: Copia el informe del batch al portapapeles.
  • Cerrar: Cierra la ventana de vista del batch.

Históricos

  • Tabla de Históricos: Muestra una lista de todos los batches anteriores con sus detalles.
  • Reiniciar: Permite reiniciar un batch seleccionado.
  • Ver: Muestra los detalles del batch seleccionado.

Ejemplo de Uso

1. Iniciar la Aplicación: Ejecuta el archivo CrawlerGPT.exe para abrir la aplicación.

2. Menú Principal: Selecciona "Nuevo Batch" para iniciar un nuevo proceso de rastreo. Selecciona "Ver Históricos" para ver el historial de rastreos anteriores. Selecciona "Salir" para cerrar la aplicación.

3. Nuevo Batch: Introduce los parámetros necesarios (URL inicial, profundidad, sitios, caracteres). Haz clic en "Iniciar" para comenzar el rastreo. La consola de batch se abrirá automáticamente para mostrar el progreso.

4. Consola de Batch: Observa el progreso del rastreo en tiempo real. Usa el botón "Copiar al Portapapeles" para copiar el contenido de la consola. Usa el botón "Cancelar Batch" para detener el proceso de rastreo.

5. Ver Históricos: Selecciona un batch del historial para ver sus detalles. Usa los botones de la tabla para reiniciar un batch o ver sus detalles.

6. Vista de Batch: Revisa los detalles del batch seleccionado. Usa el botón "Copiar al Portapapeles" para copiar el informe. Usa el botón "Cerrar" para cerrar la ventana de vista del batch.
